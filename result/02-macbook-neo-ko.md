## Product Verdict

🟡 **Buy** (조건부): MacBook Neo — A18 Pro, $599 (₩990,000). 아이폰 16 Pro 칩이 맥북에 들어갔다. 웹서핑·문서·넷플릭스? 완벽. 코딩이나 영상 편집이 목적이면? 도망가세요

🟢 **Strong Buy** (비교 대상): MacBook Air M4 — $1,099 (₩1,690,000). ₩70만원 더 내면 M4 + 16GB RAM + MagSafe + 키보드 백라이트 + Thunderbolt. 이 차이를 아끼면 후회하고, 더 내면 만족한다

## 요약

- **역사상 가장 싼 MacBook** — $599 (₩990,000). 아메리카노 180잔 가격으로 macOS 생태계 입장. "맥북이 100만원이 안 된다고?"
- **아이폰 16 Pro 칩이 맥북에** — A18 Pro 탑재. 싱글코어 ~3,400으로 M3 Air를 넘고, 멀티코어 ~8,200은 M1 수준. 빠르지만 "6코어의 한계"
- **8GB RAM 고정, 업그레이드 불가** — 2026년에 8GB라니. 탭 30개 열면 숨 막힘. 이게 이 맥북의 진짜 천장
- **빠진 것 12가지** — MagSafe·Thunderbolt·키보드 백라이트·True Tone·P3 색역 등. ₩99만원의 비밀은 "뭘 뺐는가"에 있다
- **"세컨북"의 탄생** — 카페에서 가볍게, 출장에서 간단히, 아이 숙제용으로. 메인 맥북이 아닌 "두 번째 맥북"이라는 새로운 카테고리

## 4. 누가 사야 하나 — 진지하게

### 사세요 👍

**카페 블로거 / 글 쓰는 사람**
> 글쓰기 앱 + Safari + 음악이면 충분. ₩99만원 맥북을 카페 테이블에 올려놓으면,
> 아무도 이게 Neo인지 Pro인지 모른다. 겉모습은 같으니까.

**학생 (교육용 ₩85만원)**
> 레포트 쓰고, 발표자료 만들고, 강의 듣기에 완벽.
> ₩85만원이면 학생 입장에서 **역대급 가성비**.

**세컨북이 필요한 사람**
> 이미 MacBook Pro가 있는데 침대에서 넷플릭스 볼 때, 출장 가서 메일 확인할 때.
> 무게 1.23kg에 배터리 16시간. 가볍게 들고 다니기 딱.

**아이 숙제용 / 가족 공용**
> 유튜브 숙제, 인터넷 서핑, 간단한 코딩 학습.
> iPad보다 키보드가 편하고, Chromebook보다 생태계가 좋다.

### 도망가세요 🏃

**유일한 노트북으로 쓸 사람**
> 8GB RAM은 2026년 기준으로 최소 사양. 2–3년 후에는 답답해진다.
> ₩70만원 더 내고 Air M4(16GB)를 사는 게 장기적으로 이득.

**영상 편집 / 개발자**
> Final Cut Pro, Xcode, Docker — 전부 8GB로는 버겁다.
> A18 Pro의 6코어 GPU도 M4의 10코어 GPU와는 격차가 크다.

**외장 장비를 많이 쓰는 사람**
> Thunderbolt 없음 = 고속 외장 SSD 불가. 오른쪽 USB 포트는 USB 2 (480 Mb/s).
> 모니터 연결도 한 대까지만.

**"나 맥북 샀어" 자랑이 목적인 사람**
> 카페 조명 아래에서 키보드 백라이트 안 들어오면... 아는 사람은 안다.
> (물론 대부분은 모른다. 솔직히.)

## 인사이트

**1. $599 MacBook = Apple의 "점유율 확보" 전략**

Apple은 처음으로 macOS 시장의 **입문 가격**을 $1,000 아래로 내렸다. Chromebook이 교육 시장을 장악한 동안 Apple은 $999+ MacBook Air만 고집했다. MacBook Neo는 "이제 우리도 $600대가 있어요"라는 선언이다. 마진보다 생태계 잠금(lock-in)이 목적.

**2. A18 Pro는 "재활용 칩" — 그래서 이 가격이 가능했다**

MacBook Neo의 A18 Pro는 iPhone 16 Pro에서 이미 양산된 칩이다. 새로운 M 시리즈 설계 없이 기존 iPhone 칩을 노트북에 넣었기 때문에 R&D 비용이 거의 없다. Apple이 $599를 가능하게 만든 비결은 "신칩 개발 안 함". 영리하다.

**3. "빠진 것 12가지"가 아니라 "남긴 것"이 핵심**

MagSafe·Thunderbolt·키보드 백라이트... 빠진 게 많아 보이지만, 남은 것이 더 중요하다: **macOS + A18 Pro + 16시간 배터리 + 1.23kg**. 이 네 가지만으로도 Chromebook과 Windows 저가 노트북을 압도한다. Apple이 정확히 "최소한의 맥북 경험"을 정의한 셈.

**4. 한국 프리미엄 14–17% — 저가 제품일수록 더 비싸다**

$599 × ₩1,450 = ₩868,550인데 실제 ₩990,000. 프리미엄 14%. MacBook Pro M5의 9%보다 높다. Apple이 저가 라인업에서 한국 소비자에게 더 높은 마진을 부과하는 패턴. "싸니까 좀 더 붙여도 되겠지"라는 가격 전략이 보인다.

**5. MacBook Neo의 진짜 경쟁자는 MacBook Air가 아니라 iPad다**

₩990,000 MacBook Neo vs ₩1,158,000 (iPad Air M3 ₩899,000 + Magic Keyboard ₩259,000). 비슷한 가격대에서 "키보드 달린 macOS"와 "터치스크린 iPadOS" 중 선택이다. 생산성 위주면 Neo, 콘텐츠 소비+그림 위주면 iPad. Apple이 자기 제품끼리 경쟁시키는 건 늘 하던 일이지만, 이 가격대에서는 처음이다.

## 용어집

| 용어 | 설명 |
|------|------|
| **A18 Pro** | Apple이 iPhone 16 Pro용으로 개발한 칩. 3nm(N3E) 공정, 6코어 CPU(2 Performance + 4 Efficiency) + 5코어 GPU. MacBook Neo에서 Mac 최초로 A 시리즈 칩 탑재 |
| **M4** | Apple의 Mac 전용 칩. 10코어 CPU + 10코어 GPU + 16GB+ RAM. MacBook Air, MacBook Pro 등에 탑재. A18 Pro보다 코어 수와 메모리에서 우위 |
| **sRGB vs P3** | 디스플레이가 표현할 수 있는 색 범위. sRGB는 표준, P3(DCI-P3)는 sRGB보다 25% 넓은 색역. 사진/영상 편집자에게 중요. MacBook Neo는 sRGB(추정), Air는 P3 |
| **MagSafe** | Apple의 자석 충전 커넥터. 케이블에 걸려 넘어져도 자석이 분리되어 맥북이 안전. MacBook Neo에는 없어서 USB-C 충전만 가능 |
| **Thunderbolt** | Intel/Apple이 개발한 고속 연결 규격. 최대 40 Gb/s로 외장 SSD, 모니터 등 연결. USB-C와 겉모양은 같지만 속도가 4배 빠름 |
| **Force Touch** | Apple의 감압식 트랙패드. 누르는 세기를 감지하여 다양한 동작 가능. MacBook Neo는 일반 Multi-Touch 트랙패드 사용 |
| **True Tone** | 주변 조명 색온도에 맞춰 디스플레이 색감을 자동 조절하는 기능. 눈의 피로를 줄여줌. MacBook Neo에는 미탑재 |
| **Liquid Retina** | Apple의 고해상도 IPS LCD 디스플레이 브랜드명. OLED가 아닌 LCD 기반이지만 높은 해상도와 색 정확도 제공 |
| **팬리스 (Fanless)** | 냉각 팬 없이 알루미늄 본체를 방열판으로 사용하는 설계. 완전 무음이지만 고부하 시 쓰로틀링(성능 저하) 가능성 |
| **Apple Intelligence** | Apple의 온디바이스 AI 기능. 글쓰기 도구, 이미지 생성, Siri 강화 등. 8GB RAM에서도 동작하지만 고급 기능은 제한될 수 있음 |
