## Product Verdict

- 🟡 **Buy** (조건부): MacBook Neo — A18 Pro, $599 (₩990,000). 웹·문서·넷플릭스 완벽. 코딩/편집이면 도망가세요
- 🟢 **Strong Buy** (비교): MacBook Air M4 — $1,099 (₩1,690,000). ₩70만원 더 = M4 + 16GB + MagSafe + 키보드 백라이트

## 스펙 — 아이폰이 맥북이 됐다

- **A18 Pro**: 6코어 CPU + 5코어 GPU + 16코어 Neural Engine
- **RAM**: 8GB 고정 (업그레이드 불가)
- **SSD**: 256GB ($599) / 512GB ($699)
- **디스플레이**: 13" Liquid Retina, 2408×1506, 500 nits
- **무게**: 1.23kg / **배터리**: 16시간
- **팬리스** — 완전 무음
- **색상**: Blush · Indigo · Silver · Citrus

> 256GB 모델은 **Touch ID 없음**. 512GB부터 Touch ID 포함.
> USB 포트: 왼쪽 USB 3 (10 Gb/s) / 오른쪽 USB 2 (480 Mb/s). 비대칭!

```
멀티코어:
```

> 멀티코어 = M1 수준. 6코어(2P+4E) vs M4 10코어의 한계.
> 동시 작업하면 체감 차이 확 벌어진다.

## 빠진 것 — ₩99만원의 비밀

Air M4 대비 Neo에서 빠진 것들:

**치명적** 🔴
- MagSafe 없음 → 케이블 걸리면 맥북이 날아감
- 키보드 백라이트 없음 → 어두운 카페에서 타자 못 침
- RAM 8GB 고정 → 탭 30개 열면 숨 막힘
- Touch ID 없음 (256GB만) → 비번 매번 입력

**불편하지만 참을 만한** 🟡
- Thunderbolt 없음 → USB 3 수준
- P3 Wide Color 없음 → 색감 차이
- True Tone 없음 → 자동 색온도 조절 없음
- Force Touch 없음 → 단순한 트랙패드

```
USB 포트 비대칭:
왼쪽: USB 3 (10 Gb/s)  ← 2013년
오른쪽: USB 2 (480 Mb/s) ← 2001년
외장 SSD는 반드시 왼쪽에!
```

## 인사이트

**1. $599 MacBook = Apple의 점유율 확보 전략**

Chromebook이 교육 시장을 장악하는 동안 Apple은 $999+ Air만 고집했다. Neo는 "이제 우리도 $600대가 있어요"라는 선언. 마진보다 생태계 잠금(lock-in)이 목적.

**2. A18 Pro는 "재활용 칩" — 신칩 개발 없이 $599 실현**

iPhone 16 Pro에서 이미 양산된 칩을 그대로 노트북에. R&D 비용 거의 제로. Apple이 $599를 만든 비결은 "새 칩 안 만들기".

**3. "빠진 것 12가지"가 아니라 "남긴 것"이 핵심**

macOS + A18 Pro + 16시간 배터리 + 1.23kg. 이 네 가지만으로 Chromebook과 Windows 저가 노트북을 압도. Apple이 "최소한의 맥북 경험"을 정의했다.

**4. 한국 프리미엄 14–17% — 저가일수록 더 비싸다**

$599 × ₩1,450 = ₩868,550인데 실제 ₩990,000 (+14%). M5 Pro/Max는 9–11%. 저가 제품일수록 한국에서 더 높은 마진.

**5. 진짜 경쟁자는 Air가 아니라 iPad다**

₩99만원 Neo vs ₩116만원 (iPad Air + Magic Keyboard). 비슷한 가격대에서 "키보드 macOS" vs "터치 iPadOS" 선택. Apple이 자기 제품끼리 경쟁시키는 건 처음은 아니지만, 이 가격대에서는 처음.
