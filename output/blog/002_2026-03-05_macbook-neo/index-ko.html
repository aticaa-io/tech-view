Product Verdict

🟡
Buy (조건부) — MacBook Neo — A18 Pro, $599 (₩990,000). 아이폰 16 Pro 칩이 맥북에 들어갔다. 웹서핑·문서·넷플릭스? 완벽. 코딩이나 영상 편집이 목적이면? 도망가세요
🟢
Strong Buy (비교 대상) — MacBook Air M4 — $1,099 (₩1,690,000). ₩70만원 더 내면 M4 + 16GB RAM + MagSafe + 키보드 백라이트 + Thunderbolt. 이 차이를 아끼면 후회하고, 더 내면 만족한다

요약

  • 역사상 가장 싼 MacBook — $599 (₩990,000). 아메리카노 180잔 가격으로 macOS 생태계 입장
  • 아이폰 16 Pro 칩이 맥북에 — A18 Pro. 싱글코어 ~3,400으로 M3를 넘고, 멀티코어 ~8,200은 M1 수준
  • 8GB RAM 고정, 업그레이드 불가 — 2026년에 8GB. 탭 30개 열면 숨 막힘. 이 맥북의 진짜 천장
  • 빠진 것 12가지 — MagSafe·Thunderbolt·키보드 백라이트·True Tone 등. ₩99만원의 비밀은 "뭘 뺐는가"
  • "세컨북"의 탄생 — 카페에서 가볍게, 출장에서 간단히. 메인 맥북이 아닌 "두 번째 맥북"

옵션

8GB로 할 수 있는 것:

  • ✅ Safari 탭 15–20개
  • ✅ Pages/Numbers 문서 작업
  • ✅ 넷플릭스/유튜브 스트리밍
  • ✅ 간단한 사진 편집
  • ✅ Apple Intelligence

8GB로 힘든 것:

  • ❌ 탭 30개 + Slack + 카카오톡 + 음악 동시에
  • ❌ Xcode 앱 개발
  • ❌ Final Cut Pro 영상 편집
  • ❌ 로컬 AI 모델 (LLM 등)
  • ❌ Docker 컨테이너 개발
2026년에 8GB RAM = 스마트폰 수준. "지금 당장"은 괜찮지만 2–3년 후를 생각하면 한숨이 나온다.

4. 누가 사야 하나 — 진지하게

👍 사세요

글쓰기 앱 + Safari + 음악이면 충분. ₩99만원 맥북을 카페에 올려놓으면, 아무도 이게 Neo인지 Pro인지 모른다.

세컨북이 필요한 사람
이미 MacBook Pro가 있는데 침대 넷플릭스, 출장 메일용. 1.23kg, 16시간 배터리.

아이 숙제용 / 가족 공용
유튜브 숙제, 인터넷 서핑. iPad보다 키보드 편하고, Chromebook보다 생태계 좋다.

🏃 도망가세요

유일한 노트북으로 쓸 사람
8GB RAM은 2026년 최소 사양. 2–3년 후 답답. ₩70만원 더 내고 Air M4(16GB) 추천.

영상 편집 / 개발자
Final Cut, Xcode, Docker — 전부 8GB로는 버겁다. A18 Pro 5코어 GPU vs M4 10코어 격차 큼.

외장 장비를 많이 쓰는 사람
Thunderbolt 없음 = 고속 외장 SSD 불가. 오른쪽 USB는 USB 2. 모니터 한 대까지만.

"나 맥북 샀어" 자랑이 목적
카페 조명 아래 키보드 백라이트 안 들어오면... 아는 사람은 안다. (대부분은 모르지만.)

인사이트

1. $599 MacBook = Apple의 "점유율 확보" 전략

Chromebook이 교육 시장을 장악하는 동안 Apple은 $999+ Air만 고집했다. Neo는 "이제 우리도 $600대가 있어요"라는 선언. 마진보다 생태계 잠금(lock-in)이 목적.

2. A18 Pro는 "재활용 칩" — 신칩 개발 없이 $599 실현

iPhone 16 Pro에서 이미 양산된 칩을 노트북에. R&D 비용 거의 제로. $599의 비결은 "새 칩 안 만들기". 영리하다.

3. "빠진 것 12가지"가 아니라 "남긴 것"이 핵심

macOS + A18 Pro + 16시간 배터리 + 1.23kg. 이 네 가지만으로 Chromebook과 Windows 저가 노트북을 압도. Apple이 "최소한의 맥북 경험"을 정의했다.

4. 한국 프리미엄 14–17% — 저가 제품일수록 더 비싸다

$599 × ₩1,450 = ₩868,550인데 실제 ₩990,000 (+14%). M5 Pro/Max는 9–11%. 저가 제품일수록 한국에서 더 높은 마진.

5. 진짜 경쟁자는 Air가 아니라 iPad다

₩99만원 Neo vs ₩116만원 (iPad Air + Magic Keyboard). 비슷한 가격에서 "키보드 macOS" vs "터치 iPadOS" 선택. Apple이 자기 제품끼리 이 가격대에서 경쟁시키는 건 처음.
